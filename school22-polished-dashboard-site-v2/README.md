# Школа №22 — доработанный профессиональный сайт рейтинга

## Добавлено

- Логотип школы №22.
- Более законченный официальный дизайн.
- Направления рейтинга прямо в карточках классов.
- Расширенная карточка класса.
- Radar-график по направлениям.
- Автослайд-шоу после 10 секунд бездействия.
- Любое движение мыши, клик или касание возвращает обычный режим.

## Важно

Пока API не отдаёт отдельные баллы по категориям, сайт аккуратно рассчитывает демонстрационное распределение по направлениям на основе общего балла класса.

Когда добавим в API отдельные категории по классам, заменим эту часть на реальные данные.


## v2
